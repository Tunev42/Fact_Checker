МОДУЛИ:

1)Регистрация - Нужно для создания нового аккаунта в системе

Входные данные:
  
login(Имя пользователя)
password(Пароль пользователя)

Выходные данные - Создание нового аккаунта в системе

2)Вход в систему - Аутентификация пользователя

Входные данные:

login(Имя пользователя)
password(Пароль пользователя)

3)Проверка пароля - оценка сложности пароля при регистрации

Входные данные:

password(Пароль пользователя)

Выходные данные:

"Сильный", "Средний" или "Слабый"

4)Проверка фактов - координация всего процесса проверки текста

Входные данные:

text - Текст для проверки
user_id - Для статистики

Выходные данные:

Отфарматированые результат проверки

5)Выхода из системы - Завершение сессии пользователя

Входные данные:

session(текущая сессия)

Выходные данные:

Очищение сессии и возвращении на главное меню

6)Извлечение утверждения из текста - разбивка длинного текста на отдельные фактологические утверждения

Входные данные:

text - Первоначальный текст

Выходные данные:

Список строк (каждая отдельное утверждение)

7)Поиск источников - Поиск в интернете информации по утверждению

Входные данные:

claim - Утвердление для поиска
options — Настройки поиска (количество результатов язык и т.д.)

Выходные данные:

Список найденных источников с URL заголовком и текстом

8)Проверка утверждения - Анализ источников и вынесение вердикта по утверждению

Входные данные:

claim — Утверждение
sources — Найденные источники

Выходные данные:

Словарь с вердиктом, уверенностью и объяснением

9)Извлечение домена из URL - Извлечение чистого доменного имени из полного URL

Входные данные:

url — Полный URL например, "https://www.nasa.gov/news/index.html")

Выходные данные:

Домен — например, "nasa.gov"

10)Класификация источников - Определение типа источника (научный, псевдонаучный, новостной и т.д.)

Входные данные:

source — Информация об источнике

Выходные данные:

Категория источника и оценка надежности

11)форматирование результатов

Входные данные:

results — Результаты проверки каждого утверждения

Выходные данные:

Отформатированная строка для отображения пользователю
